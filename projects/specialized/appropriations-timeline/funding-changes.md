## Methodology

This analysis was generated by parsing XML files of enacted Washington State budget bills:

- **Operating Budgets:** ESSB 5693 (2021-23 Supp), ESSB 5187 (2023-25), ESSB 5950 (2023-25 Supp), ESSB 5167 (2025-27)
- **Capital Budgets:** ESSB 5200 (2023-25), ESSB 5195 (2025-27)

**Definitions:**
- **Growth:** Agencies with >5% budget increase between consecutive biennia
- **Cuts:** Agencies with >5% budget decrease between consecutive biennia
- **Stable:** Agencies with changes within ±5%
- **New Programs:** Agencies/programs receiving funding in a biennium that had no funding in the previous biennium
- **Discontinued:** Agencies/programs that had funding in one biennium but none in the next

**Note:** Supplemental budgets adjust the base budget. Changes shown reflect enacted appropriations at the time of bill passage.
